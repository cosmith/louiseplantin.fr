# louiseplantin.fr

Run `python generate.py` to regenerate assets (reads original images from
`originals/`, writes thumbnails to `public/thumbs/` and metadata to
`src/images.json`; `originals/` is kept out of `public/` so the full-size
files are not included in the deployed build)

Run `yarn start` to launch dev server

Run `yarn deploy` to deploy on github pages
